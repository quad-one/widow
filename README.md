# Widow Frame

A open source, super light FPV Drone Frame.
Recommended are 110X motors with M2 mounting holes that are 9mm diagonally apart from each other.


# Specifications

Wheelbase: 100 or 110mm<br>
Bottom plate: 2mm<br>
